<h1 align="center">Full Stack AI Car Marketplace with NextJS, Tailwind CSS, Supabase, Prisma, Clerk, ArcJet, Shadcn UI</h1>

### Core Features:

- AI image-based car search

- Advanced filtering (make, model, price, etc.)

- Interactive EMI calculator

- Real-time test drive booking

- Admin dashboard with analytics and inventory tools

- AI-based car detail extraction from images

- Fully responsive UI using Next.js and ShadCN UI

### Make sure to create a .env file with following variables -

```shell
DATABASE_URL=
DIRECT_URL=

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
NEXT_PUBLIC_CLERK_AFTER_SIGN_IN_URL=/onboarding
NEXT_PUBLIC_CLERK_AFTER_SIGN_UP_URL=/onboarding

GEMINI_API_KEY=

ARCJET_KEY=
```

### Run this app locally

```shell
npm run build
```

### Start the app

```shell
npm run dev
```
